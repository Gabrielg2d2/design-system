# Gabs UI

Gabs UI is a simple and easy to use UI library for the Godot Engine.
It is also designed to be easy to customize and to be able to be used in any project.

## Features

- Simple and easy to use
- Easy to customize
- Easy to use in any project
- Easy to extend
- Easy to add new widgets
